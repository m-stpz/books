# Non-Functional Requirements | The core `-ities` of Modern Architecture

- Scalability
  - System capacity to handle a bigger load without losing performance
  - Latency / Performance

- Reliability
  - Handle/recover from failure
  - Avoid single point of failure
  - Error handling
  - Fault tolerance
  - Redundance
  - Availability/Uptime

- Maintainability
- Security
