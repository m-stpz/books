# Multi-leader replication topologies

- What's a replication topology?
  - Describes path through which writes are propagated through the nodes
  - With 2 leaders

```
leader1 -> leader2
leader2 -> leader1
```

However, with <= 3 leaders, various topologies are possible

- circular: similar to a linked list
  - a nodes receives from one node and forwards it to only one another node

```
        node1
        /    \
    node2    node4
        \    /
        node3
```

- star: a node is in the center and forwards its writes to all other nodes
  - similar to a tree structure

```
        node1
          |
        node4
       /    \
    node2    node3
```

- all-to-all: every leader sends its writes to every other leader

```
         node1
        /  |   \
       |  node4   |
       | /    \   |
    node2  -  node3
```
